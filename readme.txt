/********** Features of app **********/
// Frontend
1. Create,Read,Update,Delete(CRUD) Tasks
2. Used React Hook from with validation
3. Used Redux RTK Query for managing state & http request.
4. TaskLists page,TaskAdd page.
5.Responsive ui across all devices.
6. Use tailwindcss for beautiful & clean styling.

// Backend
1. Setup own server with express.js
2. Adding route,controllers.models
3. Proper error handling at global level
4. Testing api with postman
5. Clean code of api


/##### Steps to run Client(Frontend)
Step 1. Go to folder Frontend.
Step 2. npm install
Step 3. npm run dev


/##### Steps to run Backend(Server)
Step 1. Go to folder server.
Step 2. make .env file and add your MONGO_URL there.
Step 3. npm install
Step 4. npm run dev
